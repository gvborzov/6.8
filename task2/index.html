console.log

Пример использования команды console.log. При выполнении команды в консоли мы увидим текст в кавычках. 

    console.log('Привет мир!')

Alert()

Пример использования команды alert. При выполнении команды в браузере откроется всплывающее окно с указанным в кавычках текстом. 

    alert('Внимание, я всплывающее сообщение! Нажми "ОК", чтобы скрыть меня.')

Prompt()

Пример использования команды prompt. При выполнении команды в браузере откроется всплывающее окно с возможностью ввода текста. 

    prompt('Впиши текст')
